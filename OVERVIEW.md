# Bilim OS — обзор продукта

> Система управления частной школой: журнал, оценки, расписание, аналитика и связь с родителями — в одном месте.
> Демо: **https://bilimos.kg** · вход одним кликом (на странице входа есть вкладки ролей: Школа / Учитель / Ученик / Родитель), демо-пароль `erudit2025`.
> Репозиторий: **https://github.com/Ulfacar/erudit** (приватный, ветка `main`).

Легенда статусов: ✅ работает в системе · 🟡 в роадмапе («Скоро», есть заглушка) · ⬜ не начато (скрыто из навигации)

---

## 1. Что это

Bilim OS — веб-платформа (ERP) для ежедневного управления школой. Закрывает рутину, которую школы сейчас ведут в бумаге, Excel и мессенджерах: журнал и оценки, расписание, посещаемость, аналитика для директора, дневник для родителей и учеников. Доступ строго по ролям — каждый видит только своё.

**Позиционирование:** мы НЕ заменяем Кундолюк (обязательный государственный реестр отчётности) — мы закрываем ежедневную работу и аналитику, которых в нём нет. Прямая интеграция с Кундолюком — в планах.

**Технологии:** Next.js (React) · PostgreSQL (Neon) · Prisma · Mantine UI · NextAuth (авторизация). Хостинг — Vercel, домен bilimos.kg (Cloudflare). Полностью онлайн, работает из браузера, адаптирован под телефон.

## 2. Для кого — 9 ролей

| Роль | Кто это | Что получает |
|------|---------|--------------|
| `super_admin` | Директор / владелец | Полный доступ, общая картина, аналитика, настройки |
| `analyst` | Аналитик | Аналитика и отчёты на уровне администрации |
| `zavuch` | Завуч | Модерация оценок, нагрузка, дескрипторы педагогов |
| `secretary` | Секретарь | Классы, ученики, педагоги, расписание |
| `teacher` | Учитель | Журнал, оценки, ДЗ, расписание, замены |
| `curator` | Куратор класса | Свой класс, ученики, посещаемость |
| `specialist` | Логопед/психолог/медик | Профильные кабинеты |
| `student` | Ученик | Дневник: свои оценки, ДЗ, посещаемость |
| `parent` | Родитель | Дневник ребёнка, успеваемость, посещаемость |

Доступ к данным изолирован: ученик и родитель видят строго только свои данные. Дополнительно у пользователя есть «звёздный уровень» (`starLevel`) для тонкой настройки прав.

## 3. Что делает — рабочий функционал ✅

### Учебный процесс и оценивание
- **Электронный журнал и оценки** — выставление оценок по предметам и периодам.
- **Двухуровневая модерация оценок** — жизненный цикл оценки: `черновик → отправлено → отмодерировано → опубликовано`. На модерацию идут только важные категории (контрольные, зачёт, триместровая, итоговая, экзамен).
- **Категории оценок** — с весами и порядком; настраиваемые типы (проверочная работа), возможность разрешить учителям создавать свои категории.
- **Шкалы оценивания** — 5-балльная, 12-балльная, 100-балльная, буквенная.
- **Журнал аудита оценок** — фиксирует кто, когда и что изменил (старое/новое значение).
- **Учебные периоды** — триместры, каникулы, карантин; активный период.

### Расписание
- **Расписание классов и учителей**, отдельное расписание учителя.
- **Расписание звонков** — слоты: урок, перемена, завтрак, обед, полдник, окончание.
- **Замены** — замена учителя на конкретную дату с причиной.

### Люди и структура школы
- **Классы** — уровни школы (диапазоны параллелей), классы, **группы внутри класса** и **перевод ученика между группами с одобрением** (заявка → одобрение/отклонение).
- **Ученики** — карточка: ФИО, дата рождения, статус (постоянный / условный / второгодник), класс, группы, родители; мед./семейные/финансовые данные.
- **Педагоги** — карточка, предметы и часы, кураторство классов.
- **Нагрузка педагогов** — учёт часов; **передача нагрузки** (декрет/болезнь/увольнение) с read-only историей для нового учителя.
- **Дескрипторы педагога** — служебные пометки (качество работы, выговоры, повышения и т.п.) с **3 уровнями доступа** (общий / завучи / только директор), привязка к учебному году.
- **Родители** — связь родитель↔ученик, кабинет родителей.

### Посещаемость, поведение, коммуникации
- **Посещаемость** — отметки: присутствует, отсутствует, опоздал, уваж. причина, выезд, карантин.
- **Домашние задания** — выдача ДЗ по классу/предмету со сроком.
- **Срочные вопросы** — задачи/обращения с приоритетом и видимостью по ролям.
- **Происшествия** — поведение / здоровье / оборудование / безопасность; поведенческие инциденты с уведомлением родителей.
- **Новости** — школьные / для персонала / по классу.
- **Чаты** — обмен сообщениями (с вложениями).
- **Дневник** — кабинет ученика и родителя: оценки, ДЗ, посещаемость (только своё).

### Аналитика и отчёты
- **Аналитика школы** — средняя успеваемость по классам, посещаемость, рейтинг учителей, ученики с низкой успеваемостью, динамика по периодам.
- **Отчёты** — по успеваемости и по посещаемости.
- **Главная (дашборд)** — сводка: всего учеников/классов, посещаемость, средний балл, кол-во учителей, срочные вопросы, происшествия, средний балл по классам.

### Кабинеты специалистов
- **Логопед**, **Психолог**, **Медкабинет**, **кабинет работы с родителями**.
