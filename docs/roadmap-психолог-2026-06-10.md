## 3. Что построено — по блокам ТЗ (всё ✅ в проде)

### Блок 1. Безопасность и этика
- **RLS-изоляция**: психолог видит только свои кейсы + принятые co-доступы (`src/shared/lib/psy-scope.ts`).
- **Анонимность руководству**: дашборд `/psychologist/overview` + `/api/v1/psy/dashboard` — только агрегаты (по статусам, рискам, классам, % улучшений), без ФИО и текстов.
- **NER-обезличивание для AI**: ФИО→`[УЧЕНИК_N]` на сервере ДО облачной LLM, карта маркеров не уходит (`src/shared/lib/ai/psy/deidentify.ts`, кириллица через `\p{L}`). Ре-идентификация ответа — на сервере.

### Блок 2. Управление кейсами
- **Статусы**: Новый / В работе / Приостановлен / Закрыт.
- **Риски + Safeguarding**: Зелёный / Жёлтый / Красный; красный → слепой алерт координатору (UC-5).
- **PRE/POST история**: входная диагностика при поступлении (`isIntake`) навсегда привязана к ученику.

### Блок 3. Консультации + AI-ассистент (DAP)
- **Voice-to-DAP**: запись с микрофона → **локальный Whisper** в браузере (`@xenova/transformers`, `localTranscribe.ts`) — аудио не уходит в сеть.
- **Офлайн-патч**: аудио + черновик в IndexedDB (`draftStore` v2), переживают потерю сети.
- **AI-раскладка**: текст → Data/Assessment/Plan (`/api/v1/psy/sessions/dap`), fallback-сплиттер без ключа.
- **Анти-галлюцинация**: оригинал и AI-черновик раздельно; «завершить» заблокировано до галочки `is_human_verified` (серверная валидация).

### Блок 4. Бумажные тесты и проективные методики (O2O)
- **Цифровой сейф**: фото бланка/рисунка → MinIO (`/api/v1/psy/vision`).
- **Privacy-OCR**: tesseract.js находит подпись и блюрит её (клиентский canvas) до отправки в Vision (`ProjectiveTest.tsx`).
- **OMR бланков**: камера считает сырые баллы → шкалы (`omrExtract`, `/api/v1/psy/omr`).
- **Калькулятор шкалы + график**: ручной ввод → `PsyMeasurement` → график динамики (`ScoreTest.tsx`, `Dynamics.tsx`).

### Блок 5. Конструктор методик
- Старший психолог собирает тест (вопросы, шкалы 1–10, файлы, симптомы) — `/psychologist/methods`.
- **Версионирование** v1/v2 (`parentTemplateId`) + **Mapping Rule gate** (HTTP 409 при смене шкалы) — старая статистика не ломается.

## 6. Курсовая логика (из транскриптов)
- Курс ~6 сессий → исход: **improved** (закрыть), **repeat** (новый раунд, `courseRound++`), **referred** (к психиатру/логопеду, пауза).
- При исчерпании методик (repeat ≥2 / referred) — уведомление старшему психологу.
- Качественное «до/после» — поле `qualNote` на сессии.
- Встречи: первичная диагностика / плановая / экстренная / с родителями / **с учителями** / групповая.
